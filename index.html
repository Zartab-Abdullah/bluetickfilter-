How to install — 4 steps
1
Click the button above — a .zip file will download. Find it in your Downloads folder and unzip it (right-click → Extract All)
2
Open Microsoft Edge and type edge://extensions in the address bar, then press Enter
3
Turn on Developer mode using the toggle in the top-right corner of that page
4
Click Load unpacked → select the unzipped x-filter-v2 folder → the extension is now installed. Go to x.com and blue-tick tweets will disappear immediately
